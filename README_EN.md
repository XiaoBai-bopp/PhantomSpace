## 🌌 What is Phantom Space?

**Phantom Space** is a full-featured virtual environment engine for Android. It creates a completely isolated parallel space where you can clone any app and run it with superpowers — without root, without messing with your system.

Think of it as a **virtual phone inside your phone**, but lighter, faster, and packed with developer-grade tools.

---

## ✨ Key Features

### 🧬 Xposed Framework (No Root Required)
- Fully integrated Xposed framework inside the virtual space.
- Load any Xposed module directly into your cloned apps.
- No system modification, no bootloop risk.

### 🕶️ Hide Xposed & Root
- Built-in obfuscation engine hides Xposed and root traces from virtualized apps.
- Pass common detection methods used by banking apps, games, and enterprise software.
- Your virtual space looks like a clean, unmodified device.

### 🌍 Virtual Location
- Set a custom GPS location for any app individually.
- Perfect for location-based games, testing, or privacy.

### 🔐 Google Mobile Services (GMS)
- Full GMS support inside the virtual environment.
- Run Google-dependent apps without issues — push notifications, logins, and more.

### 🛡️ Daemon Process
- Enable persistent background service to keep your virtual apps alive.
- Receive notifications in real-time even when Phantom Space is closed.

### 🧩 App Cloning & Privacy
- Clone any app multiple times, each with its own isolated data and settings.
- Use separate accounts simultaneously on one device.
- Hide your real identity — apps in the phantom space know nothing about your real system.

## 🚀 Quick Start

1. **Install** the APK from the link above.
2. Open Phantom Space and grant the necessary permissions.
3. Tap the `+` button to add apps into the virtual environment.
4. For Xposed modules: go to `Modules` → enable desired modules → restart the virtual app.
5. To use Virtual Location: tap on an app → `Location` → set coordinates.
