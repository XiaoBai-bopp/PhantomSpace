## 🌌 什么是幻影空间？

**幻影空间** 是一款功能完整的 Android 虚拟环境引擎。它可以在你的手机上创建一个完全隔离的平行空间，让你克隆任意应用并赋予其超能力——**无需 ROOT，无需修改系统**。

你可以把它理解为**手机里的另一部虚拟手机**，但更轻、更快，并且内置开发者级的强大工具。

---

## ✨ 功能特性

### 🧬 内置 Xposed 框架（免 ROOT）
- 虚拟空间内完整集成 Xposed 框架。
- 直接加载任意 Xposed 模块，作用于分身应用。
- 不修改宿主系统，彻底告别卡开机、变砖的风险。

### 🕶️ 隐藏 Xposed 与 Root
- 内置混淆引擎，隐藏虚拟环境内的 Xposed 和 Root 痕迹。
- 轻松绕过银行 App、游戏、企业软件的检测。
- 让每个虚拟空间看起来都像一台纯净未修改的设备。

### 🌍 虚拟定位
- 为每个分身单独设定 GPS 坐标。
- 完美用于位置类游戏、测试，或保护你的真实位置隐私。

### 🔐 谷歌服务 (GMS)
- 虚拟环境内完整支持 GMS。
- 依赖谷歌服务的应用可正常使用推送、登录等功能，不闪退。

### 🛡️ 守护进程
- 开启持久后台服务，确保虚拟应用长久在线。
- 即使关闭幻影空间，你仍能实时收到分身的消息通知。

### 🧩 无限分身 & 隐私空间
- 任意应用多次克隆，各自数据独立，互不干扰。
- 一台手机同时登录多个账号。
- 真实身份完全隐匿——虚拟应用读取不到你的真实设备信息。

## 🚀 快速开始

1. 点击上方按钮下载并安装 APK。
2. 打开幻影空间，按引导授予必要权限。
3. 点击右下角 `+` 把应用添加到虚拟环境。
4. Xposed 模块：进入「模块管理」→ 勾选你需要的模块 → 重启虚拟应用即可。
5. 虚拟定位：长按分身图标 → 选择「位置」→ 设定经纬度。
